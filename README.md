# Phenopacket API

Express + Typescript + MongoDB stack for building a phenopackets API.

## Getting started

### Using Docker

1. `docker compose up -d`
1. Should be available on http://localhost:3000

### Local node js

1. `npm install`
1. `npm run build`
1. `npm start`
1. Should be available on http://localhost:3000

#### Development mode

1. `npm install`
1. `npm run dev`
1. Should be available on http://localhost:3000
